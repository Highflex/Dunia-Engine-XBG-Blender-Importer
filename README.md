# Dunia Engine XBG Importer for Blender

See original for full description, this is my own personal fork to work on Animation format research/functionality.

Currently the goal is to have full animation support for Far Cry 2 across every known .MAB file plus weapon parts.

Current Changes:
- Root Motion ( Position / Rotation )
- Translation Tracks for Bones
- Additional Motion Tracks for Weapon Parts ( Still in progress )
- Helpers to create First-Person Camera
